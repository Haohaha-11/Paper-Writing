# cspapers Query Guide

Use cspapers.org as title/abstract-level retrieval evidence.

## Query Families

Create at least these query families:

- exact working title;
- method + task;
- dataset + task;
- central abstract claim;
- baseline + task;
- key technical phrase;
- acronym expansion and acronym-only query.

## Venue Sets

```text
CVPR,ICCV,ECCV,ICLR,ICML,NeurIPS
```

NLP:

```text
ACL,EMNLP,NAACL,ICLR,ICML,NeurIPS
```

```text
NDSS,Usenix,SP,CCS,ACSAC,ASIACCS,EUROSP
```
