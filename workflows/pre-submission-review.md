# Pre-Submission Review Workflow

## Goal

Find serious rejection risks before submission.

## Steps

1. Read paper context, target venue, and available draft.
2. Run `venue-compliance-review.md`.
3. Run `claim-evidence-auditor`.
4. Run `novelty-scout`.
5. Run `paper-redteam`.
6. Run section-specific skills where evidence or clarity risks are found.
7. Run `revision-editor`.
8. Rank issues by decision impact and fix effort.
9. Create final action list.

## Output

Use `examples/review-report-template.md`.
