# arXiv Export Checklist

Status: source-backed draft  
Last checked: 2026-07-15

- [ ] Include all local `.sty`, `.cls`, and macro files required to compile.
- [ ] Include `.bib` and/or `.bbl` as required by the final source package.
- [ ] Remove hidden files, local build artifacts, and editor metadata.
- [ ] Do not submit a referee-mode version.
- [ ] Include glossary, index, or generated files if the TeX source needs them.
- [ ] Ensure figure paths are local and case-correct.
- [ ] Compile from the submitted source package before upload.

## Source

- arXiv TeX submission help: https://info.arxiv.org/help/submit_tex.html
