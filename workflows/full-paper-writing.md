# Full Paper Writing Workflow

## Goal

Move from idea/results to a venue-aware draft plan.

## Steps

1. Fill `examples/paper-context-template.md`.
2. Run `paper-strategist`.
3. Build initial claim-evidence matrix.
4. Run `novelty-scout` before using strong novelty claims.
5. Draft Abstract and Introduction with `abstract-writer` and `introduction-writer`.
6. Draft Related Work using `related-work-writer` and verified citations.
7. Draft Methods with `method-writer`.
8. Draft Experiments with `experiments-writer`.
9. Draft Results with `results-writer`.
10. Draft Discussion with `discussion-writer` when the paper needs a distinct interpretation section.
11. Draft limitations, ethics, conclusion, and appendix with the corresponding section skills.
12. Run `figure-caption-writer` on figures and tables.
13. Run `claim-evidence-auditor`.
14. Run `venue-compliance-review.md`.
15. Run `paper-redteam`.
16. Revise by impact and effort using `revision-editor`.
17. Compress only after major evidence and compliance risks are resolved.

## Output

- section outline;
- contribution map;
- claim-evidence matrix;
- risk list;
- next draft actions.
