# Camera-Ready Workflow

## Goal

Prepare accepted papers without introducing policy or formatting errors.

## Steps

1. Switch from anonymous to camera-ready metadata.
2. Restore acknowledgments only when allowed.
3. Apply accepted-paper template if different from submission template.
4. Incorporate reviewer-requested changes.
5. Check author list, affiliations, funding, code/data links, and appendices.
6. Compile from a clean checkout.
7. Archive sources and PDF.

## Rule

Do not silently update templates. Record source and checksum changes.
