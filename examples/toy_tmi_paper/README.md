# Toy TMI Paper Example

Synthetic example for an IEEE TMI-style medical-imaging manuscript.

## Scenario

Target venue: IEEE TMI rolling  
Paper type: journal manuscript  
Topic: uncertainty-aware lesion segmentation with multi-site validation  

## Files

- `paper-context.md`: medical-imaging paper context.
- `medical-risk-check.md`: TMI-style privacy, validation, and clinical-scope checks.
- `venue-checklist-run.md`: example compliance notes.

The example intentionally avoids real patient data, clinical claims, or benchmark numbers.
