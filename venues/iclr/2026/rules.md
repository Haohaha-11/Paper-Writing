# ICLR 2026 Rules

Status: source-backed draft  
Last checked: 2026-07-15

## Core Rules

- Submission review is double blind.
- Main submission text limit: 9 pages.
- Rebuttal/camera-ready main text limit noted in official guide: 10 pages.
- References and appendices are outside the main page limit.
- Use official ICLR template.
- Significant LLM use must be disclosed when required by official policy.
- Do not include author names, acknowledgments, or identifying metadata in anonymous submission.

## Writing Implications

- Put the core method and evidence inside the main limit.
- Avoid hiding central claims in the appendix.
- Keep contribution bullets concrete and testable.
- Make limitations/reproducibility statements easy to find.
