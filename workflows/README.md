# Workflows

Workflows compose skills and venue rules into repeatable paper-writing processes.

Recommended order for a new paper:

1. `draft-to-submission.md`
2. `full-paper-writing.md`
3. `section-rewrite.md`
4. `venue-compliance-review.md`
5. `pre-submission-review.md`
6. `cspapers-assisted-review.md`
7. `rebuttal.md`
8. `camera-ready.md`
9. `arxiv-export.md`

Recommended order for an existing draft:

1. `review-existing-draft.md`
2. `venue-compliance-review.md`
3. `pre-submission-review.md`
4. `section-rewrite.md`
5. `camera-ready.md` or `rebuttal.md`
