# IEEE TMI Checklist

- [ ] Correct paper type selected.
- [ ] Page limit checked for paper type.
- [ ] PDF below 40 MB.
- [ ] Figures and tables are in the main text.
- [ ] No author biographies.
- [ ] Affiliations, emails, and support acknowledgments follow IEEE first-page footnote conventions.
- [ ] Conference extension is disclosed if applicable.
- [ ] Prior conference paper uploaded as supporting material if applicable.
- [ ] Dataset provenance is clear.
- [ ] Privacy, consent, IRB, or de-identification language is included where applicable.
- [ ] Validation protocol matches claim strength.
- [ ] Limitations discuss generalization and clinical workflow when relevant.
