# Full Paper Writing Workflow

## Goal

Move from idea/results to a venue-aware draft plan.

## Steps

1. Fill `examples/paper-context-template.md`.
2. Run `paper-strategist`.
3. Build initial claim-evidence matrix.
4. Run `novelty-scout` before using strong novelty claims.
5. Draft Abstract and Introduction.
6. Draft Related Work using verified citations.
7. Draft Methods and Experiments from actual evidence.
8. Run `claim-evidence-auditor`.
9. Run `paper-redteam`.
10. Revise by impact and effort.

## Output

- section outline;
- contribution map;
- claim-evidence matrix;
- risk list;
- next draft actions.
