# Toy ML Paper Walkthrough

This folder is a synthetic example that demonstrates how to use the repository. It is not a real paper and does not contain real benchmark results.

## Scenario

Target venue: ICLR 2026  
Paper type: method paper  
Topic: uncertainty-aware segmentation for synthetic medical images  

## Files

- `paper-context.md`: filled paper context.
- `claim-evidence-matrix.md`: claim audit example.
- `cspapers-hit-classification.md`: novelty-scout style search-output example.
- `redteam-report.md`: pre-submission review example.
- `rebuttal-plan.md`: response-planning example.

## How To Read

1. Start with `paper-context.md`.
2. Use `skills/paper-strategist/SKILL.md` to inspect the thesis and contribution map.
3. Use `claim-evidence-matrix.md` to see which claims are safe or risky.
4. Use `cspapers-hit-classification.md` to see how novelty risk should be reported.
5. Use `redteam-report.md` to see a pre-submission review output.
6. Use `rebuttal-plan.md` to see how review concerns become revision actions.

## Important

All paper titles, results, and reviewer comments in this folder are synthetic examples for workflow demonstration.
