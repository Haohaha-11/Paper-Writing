# Venues

Venue files store official-source rules and checklists. Rules change yearly, so annual conference rules should live under `venues/<venue>/<year>/`. Rolling journal or platform rules should use `rolling/`.

## v0.1 Coverage

- detailed first pass: ICLR 2026, CVPR 2026, IEEE TMI rolling, arXiv rolling;
- source-backed stubs: NeurIPS 2026, ICML 2026, ECCV 2026, AAAI-26.

## Rule

Third-party summaries are useful hints, not authority. Page limits, anonymity, checklist requirements, AI/LLM disclosure, and dates must be checked against official venue or publisher pages before enforcement.
