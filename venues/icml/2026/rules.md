# ICML 2026 Rules

Status: source-backed stub  
Last checked: 2026-07-15

## Initial Rules

- Main body limit: 8 pages.
- References and appendices are in the same PDF but outside the main body count.
- Submissions are LaTeX-only.
- Submission PDF size limit noted in official instructions: 50 MB.
- Anonymous submission should not reveal author identity.
