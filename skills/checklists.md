# Skill Checklists

## Before Any Writing Skill

- [ ] Target venue and year known.
- [ ] Paper type known.
- [ ] Current draft or context available.
- [ ] Main contributions listed.
- [ ] Evidence available for main claims.
- [ ] Venue checklist reviewed.

## Before Adding Or Revising Claims

- [ ] Claim type identified.
- [ ] Evidence source identified.
- [ ] Novelty-sensitive words checked.
- [ ] Citation support verified or marked unchecked.
- [ ] Scope limitations preserved.

## Before Related Work Changes

- [ ] Existing BibTeX/local bibliography checked.
- [ ] User-provided must-cite papers preserved.
- [ ] cspapers or other search source used when appropriate.
- [ ] Missing target-venue papers flagged.
- [ ] No unsupported negative comparison added.

## Before Reviewer Simulation

- [ ] Venue rules loaded.
- [ ] Claim-evidence matrix available or generated.
- [ ] Novelty/related-work scan available or marked missing.
- [ ] Draft sections provided.
- [ ] Known limitations provided.

## Before Patch Suggestions

- [ ] LaTeX commands preserved.
- [ ] Labels/refs/citations preserved.
- [ ] Equation and figure references preserved.
- [ ] Scientific content removed is listed.
- [ ] Human approval required before applying.
