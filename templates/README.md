# Templates

This directory records template provenance and import policy. It should not become a random mirror of conference templates.

## Rules

- Prefer official venue or publisher sources.
- Record URL, retrieval date, commit or artifact version, and checksum.
- Keep pristine upstream files separate from examples.
- Do not overwrite a paper's locked template without explicit user approval.
- Verify license or redistribution terms before committing template files.

## Files

- `registry.md`: known official template sources.
- `import-policy.md`: import rules.
- `import-record-template.md`: required provenance record format.
- `import-records/`: actual import records.
- `checksum-policy.md`: checksum and update policy.
- `import-log.md`: current import status.
- `checksums.sha256`: sha256 for imported files.
- `compile-validation.md`: template compile verification results.
- `upstream/`: imported pristine upstream template files.
