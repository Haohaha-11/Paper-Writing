# Template Import Policy

Before committing a template file:

1. Confirm the source is official.
2. Record retrieval date.
3. Record commit hash, release version, or artifact URL.
4. Compute sha256 for imported files.
5. Record license or redistribution status.
6. Keep pristine files separate from edited examples.
7. Document any project-local modifications.

If license or redistribution status is unclear, reference the source instead of copying the template.
